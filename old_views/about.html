About the USDOT Secure Data Commons Platform

The USDOT Secure Data Commons (SDC) is a cloud-based analytics platform that enables traffic engineers, researchers, and data scientists to access transportation-related datasets. The U.S. Department of Transportation (USDOT) created the SDC to provide a secure platform for sharing and collaborating on research, tools, algorithms, and analysis involving sensitive datasets using commercially available tools, without needing to install tools or software locally.

What It Does

  • Enables scalable data storage and analysis and user access protocol via cloud-based platforms
  • Leverages cloud capabilities to share complex (high volume, velocity, or variety) transportation datasets with the transportation research community
  • Provides authorized access to users through a data use agreement with revocable access terms to protect the sensitivity of the data
  • Provides users with predefined tools for data analysis and encourages the creation of custom toolsets, open sharing of code, and addition of datasets among the user community
  • Ensures that sensitive data are protected through adherence to USDOT’s security standards for information technology

Core Capabilities

  • Granular access controls for data and code for users
  • Robust security architecture that secures datasets and user workstations
  • Ability for users to upload and share their own data, create code, and use analytic tools in the sandbox
  • Ability to create and curate high-value datasets that are easily searchable and linkable
