Project Managers

Managing and sharing transportation research data, securely, is now easier than ever.

The USDOT Secure Data Commons (SDC) is an access-controlled, cloud-based data environment that enables users to conduct analyses and develop and test new tools and software products. The SDC stores sensitive transportation data made available by participating data providers and grants access to approved researchers to work with these datasets. The SDC also provides access to open-source tools and allows researchers to collaborate and share code with other system users. It is not intended to be an alternative to any local jurisdiction’s traffic management center or local data repository.

User Costs

Several factors can affect cost estimations when determining whether to use the SDC platform or create a customized solution. The SDC Starter Kit includes a cost estimator spreadsheet to help managers account for these factors and estimate the cost of operating the SDC from the perspectives of the platform, project, and user.

Project managers should evaluate costs from all three standpoints to assess their needs. Costs are provided in three categories:

  • Cloud consumption
  • Information technology services
  • Enablement

Starting a Project

Onboarding steps vary based on each project’s needs. To help, the SDC Starter Kit offers a variety of training and onboarding materials. Typically, the SDC admin team works with the project owner to introduce the data providers and analysts to the SDC and get your project up and running.

Creating a New SDC

Several resources are available to assist USDOT project managers in developing your own SDC-like solution. The first step is to visit the SDC Starter Kit page and review the following resources:

  • Terraform Templates

    The Terraform Templates link to the public repository for the SDC and come with a README file on how to run the templates to quickly set up the infrastructure on which the SDC resides in a way that meets the project’s needs.

  • SDC Administrator Training Modules

    The SDC Administrator Training modules are continually updated with training videos on operational aspects of maintaining an SDC, like creating a user in Active Directory, navigating the AWS management console, and onboarding users. Project owners can tailor these standard operating procedures to meet their needs.
